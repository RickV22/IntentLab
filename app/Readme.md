# IntentLab

Aplicación móvil desarrollada como proyecto académico enfocada en la gestión de usuarios y el manejo de datos locales en Android.

---

## Descripción

IntentLab es una aplicación que permite registrar usuarios, visualizar un historial de registros y gestionar dicha información de forma eficiente. Además, integra funcionalidades para compartir datos mediante archivos PDF utilizando Intents.

El proyecto implementa almacenamiento local estructurado con Room y gestión de preferencias con DataStore, junto con la comunicación entre componentes de Android.

---

## Tecnologías utilizadas

- Kotlin
- Android Studio
- Room Database
- DataStore
- Intents
- Generación de PDF (Android)

---

## Funcionalidades

- Registro de usuarios
- Visualización de historial de usuarios registrados
- Eliminación de usuarios desde el historial
- Exportación del historial a PDF
- Compartir PDF a otras aplicaciones mediante Intents
- Persistencia de datos con Room
- Gestión de preferencias con DataStore

---

## Objetivos
Este proyecto fue desarrollado con fines educativos para fortalecer habilidades en:

Desarrollo de aplicaciones Android
Manejo de almacenamiento local
Uso de Intents para compartir información
Gestión de datos persistentes
